# Manga-style-character-portrait-sprites-
Sprites in my manga art-style made to look similar to that of sprites in the Super Famicom era. 


HINT:
For best edits, I provided the split frames version, i made with Godot in mind, so my "secret" to making customizeable/easily editable characters is just layering them and controlling them with individual IDs linked to the
animation names. For easy edits, just change the hair/face/shirt individually, that way you can make a bunch of sprites in the same consistent style.
